Dataset Used: Iris Dataset

Source:
Scikit-learn Built-in Dataset

Description:
The Iris dataset contains 150 samples of iris flowers with 4 features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Classes:
- Setosa
- Versicolor
- Virginica

This dataset is loaded directly using sklearn.datasets.load_iris().
