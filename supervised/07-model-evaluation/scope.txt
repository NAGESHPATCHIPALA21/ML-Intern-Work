MACHINE LEARNING MODEL EVALUATION

==================================================

1. OVERVIEW
   ==================================================

Model Evaluation is the process of measuring how well a machine learning model performs on unseen data.

It helps determine whether a model generalizes effectively and assists in selecting the best model.

3. SUPERVISED LEARNING EVALUATION
=================================

A. Accuracy

Formula:

Accuracy = (TP + TN) / (TP + TN + FP + FN)

Advantages:
• Easy to interpret.

Limitations:
• Can be misleading for imbalanced datasets.

---

B. Precision

Formula:

Precision = TP / (TP + FP)

---

C. Recall

Formula:

Recall = TP / (TP + FN)

---

D. F1 Score

Formula:

F1 = 2 × (Precision × Recall) / (Precision + Recall)

===================

TP → True Positive

FN → False Negative

==================
